Normalization Mathematics

For each channel \( c \in \{R, G, B\} \):

\[ x_c^{\text{norm}} = \frac{x_c^{\text{raw}} - \text{MEAN}_c}{\text{STD}_c} \]

Where \( \text{MEAN} \approx 0.49 \) and \( \text{STD} \approx 0.20 \).

3. Model Architecture

ResNet-50

Pre-trained on ImageNet

Architecture Modification

Adapted for 32x32 input: Conv1 stride 2→1, Remove MaxPool, FC head modified.

\[ \text{fc}: \mathbb{R}^{2048} \rightarrow \mathbb{R}^{10} \]

Where 2048 is the original feature dimension.

Loss Function: Cross-Entropy

\[ \mathcal{L} = -\frac{1}{N}\sum_{i=1}^{N}\sum_{c=1}^{10}y_{i,c}\log(\hat{y}_{i,c}) \]

Where:

  • \( N \) = batch size
  • \( y_{i,c} \) = one-hot encoded true label
  • \( \hat{y}_{i,c} \) = predicted probability
  • Label Smoothing = 0.1

Optimizer: SGD

\[ \theta_{t+1} = \theta_t - \eta \cdot \nabla_{\theta}\mathcal{L} \]

Where:

  • \( \eta = 0.1 \) (learning rate)
  • Momentum = 0.9
  • Weight Decay = 5e-4
  • Scheduler: CosineAnnealingLR

4. Training Algorithm

Forward Pass

\[ \hat{y} = f_{\theta}(x) \]

Model prediction

Loss Computation

\[ \mathcal{L} = \text{CrossEntropy}(\hat{y}, y) \]

Calculate error

Backward Pass & Update

\[ \theta \leftarrow \theta - \eta \nabla_{\theta}\mathcal{L} \]

Update parameters

Evaluation Metrics

Accuracy

\[ \text{Acc} = \frac{\text{Correct}}{\text{Total}} \]

Precision (macro)

\[ P = \frac{1}{10}\sum_{c=1}^{10}\frac{TP_c}{TP_c + FP_c} \]

Recall (macro)

\[ R = \frac{1}{10}\sum_{c=1}^{10}\frac{TP_c}{TP_c + FN_c} \]

F1-Score

\[ F1 = 2 \cdot \frac{P \cdot R}{P + R} \]
